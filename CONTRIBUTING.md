# Contributing

Contributions are welcome!

## Guidelines

1. **Berlin focus**: Projects must be specifically about Berlin or have a significant Berlin component
2. **Data-driven**: Projects should involve data analysis, visualization, or interactive elements
3. **Quality**: Include only high-quality, impactful journalism
4. **Format**: Follow the existing format with emoji, title, link, date, and description

## How to add an entry

1. Fork this repository
2. Add your entry in the appropriate section
3. Use this format:
   ```markdown
   ### [Title](URL)

   **Date Â· Source**

   Brief description of the project and its methodology.
   ```
4. Create a Pull Request

## Entry format

- Use an appropriate emoji that represents the topic
- Link to the original project
- Include publication date
- Add a concise description (2-3 sentences)
- Mention data sources if notable

## Questions?

Open an [Issue](../../issues) if you have questions or suggestions.
